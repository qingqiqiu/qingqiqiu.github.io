My Java Leaning dinary

** JAVA 杂乱知识点

int————4byte——–[-2^31,2^31-1]

  • 关系运算优先级小于一般运算，==和！=优先级比其他关系运算符低，连续的关系运算从左到有进行。
    单目运算优先级大于双目运算
    Alt + / 推荐可能内容

    判断两个浮点数用（Math。abs（a-b）<1e-6）

  • OUT: ————- break OUT; 跳出指定循环
    break · continue可以跳出标记循环
    continue main_LOOP;

  • boolean类型 只能赋值为TRUE， FALSE ，判断时直接

  • int [] a=new int[100]定义数组，会得到默认的0值，a.length可以得到数组数的个数
    数组变量是数组的管理者而非数组本身
    复制数组必须遍历数组将数逐个拷贝

  • for(int k : data),data[]是一个数组，用于遍历数组，每一轮，k等于data的值，从0到n
    Math.sqrt();开平方
    char c=’A’;则，c++为‘B’;
    char d=(char)(c+’a’-‘A’);//大小写转换

    逃逸字符：由一个‘\’开头，后面跟上另一个字符，组合成一个字符，用来表达无法印出来的控制字符或特殊字符；

  • Intiger.isDigit(‘ ‘);//判断是不是整数
    Character.isLawerCase(‘’);//判断是不是小写
    Character.toLawerCase(‘’);//转换成小写

    abs();绝对值 pow();幂次 random(); 随机数 round();四舍五入得到整数

    String s=in.next();不接受空格
    String s=in.nextline();接受空格

    input.equals(“scsh”);判断输入的是不是scsh
    t.equals(“scsh”);判断字符串t是不是scsh

    String s;
    s.charAt(3);输出3位置上的字符
    s.substring(2);从第2个开始输出
    s.substring(2,5);从第2个数字输出至第5 个之前的字符

  • s.startswith(t)；字符串是否以t开头
    s.endwith(t);字符串是否以t结尾
    s.trim(); 删掉字符串两端空格
    s.replace(c1,c2); 字符串中所有c1变成c2
    s.toLowerCase() s.toUpperCase();把字符串中所有字符全变大、小写
